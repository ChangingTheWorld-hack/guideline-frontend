# treatment-guidelines

Для запуска проекта необходимо установить на компьютер среду разработки `Node.js` и пакетный менеджер `yarn`.

Далее нужно:

1. Зайти в папку /frontend.
2. Выполнить команду

```bash
$ yarn & yarn dev
```

Для получения prod-версии нужно выполнить команду:

```bash
$ yarn & yarn build
```
